# 무인도 여행

## 문제정보

- 사이트 : pg (프로그래머스 코딩테스트 사이트)
- 문제 번호 : 154540
- 문제 분류 : Dfs
- 난이도 : level 2

## 문제 풀이

### 생각한 문제 조건

   > **💬 생각 노트**
   > 기본적인 dfs 구조에 count를 가지고 다니며 결과를 저장하는 방법을 추가하였다.

### 내가 짠 코드

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class PG154540 {

    private static final int[][] d = {{0,1},{0,-1},{1,0},{-1,0}};

    public static void main(String[] args) {
        String[] map = {"X591X","X1X5X","X231X", "1XXX1"};
        System.out.println(Arrays.toString(solution(map)));
    }

    public static int[] solution(String[] maps) {
        int[] answer = {};
        List<Integer> list = new ArrayList<>();
        boolean[][] visit = new boolean[maps.length][maps[0].length()];
        for (int i = 0; i < maps.length; i++) {
            for (int j = 0; j < maps[i].length(); j++) {
                if(!visit[i][j] && maps[i].charAt(j) != 'X'){
                    visit[i][j] = true;
                    list.add(dfs(i, j, maps, visit, maps[i].charAt(j) - '0'));
                }
            }
        }
        if(list.isEmpty()) {
            answer = new int[]{-1};
        } else {
            answer = list.stream().mapToInt(Integer::intValue).sorted().toArray();
            }
        return answer;
    }

    public static int dfs(int x, int y, String[] maps, boolean[][] visit, int count){

        for (int i = 0; i < 4; i++) {
            int nx = x + d[i][0];
            int ny = y + d[i][1];

            if(nx < 0 || nx >= maps.length || ny < 0 || ny >= maps[0].length()) continue;
            if(!visit[nx][ny] && maps[nx].charAt(ny) != 'X') {
                visit[nx][ny] = true;
                count = dfs(nx, ny, maps, visit, count+(maps[nx].charAt(ny) - '0'));
            }
        }
        return count;
    }
}
```
